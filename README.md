## Installation
! Please Note : Required libraries : **glfw3** and **OpenGL**.

```ssh
# Clone this repository
git clone git@github.com:baptistejouin/the-imac-light-corridor.git

# Change directory
cd the-imac-light-corridor

# init CMake
cmake .

# Complile and run
make run
```
